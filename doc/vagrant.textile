h3. Using Vagrant

This could be more automated but current here are the steps:
gem install vagrant
vagrant up
vagrant ssh
sudo gem install bundler
cd /vagrant
bundle install
ruby config/setup.rb -n 'Cross Project Portal' -D xproject -u root -p '' -t xproject -y -q -f
RAILS_ENV=production rake db:migrate:reset
RAILS_ENV=production rake app:setup:new_app
